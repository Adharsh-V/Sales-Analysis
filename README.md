# Sales Data Analysis & Dashboard Project

### * Project Overview

This project demonstrates an end-to-end Sales Data Analysis pipeline, where raw and inconsistent data was transformed into a clean, structured, and insightful dataset using Excel, SQL, and Power BI.

### Tools & Technologies
Microsoft Excel – Initial data exploration and preprocessing
SQL – Data cleaning, transformation, and validation
Power BI – Dashboard creation and data visualization

### Project Workflow
1. Data Understanding & Exploration
Imported raw sales dataset
Analyzed data structure, relationships, and quality issues
Identified inconsistencies across multiple dimensions
2. Data Cleaning & Preprocessing

Performed extensive data cleaning to handle real-world data challenges:

### Handling Data Inconsistencies
Customer Data Issues
Resolved cases where the same Customer ID was linked to multiple customer names
Standardized and corrected records to maintain uniqueness
Product (Item) Data Issues
Fixed scenarios where the same Item ID corresponded to different items
Ensured accurate mapping between Item ID and Item Name
Warehouse Data Issues
Cleaned inconsistencies where Warehouse IDs mapped to different warehouse records
Maintained one-to-one relationships
### Text Standardization
Removed special characters from:
Item names
Country names
Ensured consistent formatting for better analysis and reporting
### General Data Cleaning
Removed duplicates and null values
Standardized column formats
Ensured referential integrity across tables
3. Data Transformation (SQL)
Designed structured datasets for:
Customers
Products
Orders
Warehouses
Implemented:
Joins across multiple tables
Aggregations for KPI calculations
Filtering and conditional logic
Derived metrics such as:
Total Revenue
Total Orders
Quantity Sold
4. Data Visualization (Power BI)

Developed an interactive and user-friendly dashboard featuring:

* Sales Trend Analysis
* Top Performing Customers & Products
* Country-wise Sales Insights
* Warehouse Performance Analysis
* Order Fulfillment Tracking
* Key Performance Indicators (KPIs)
* Key Insights
Identified top revenue-generating products and customers
Detected data inconsistencies affecting business accuracy
Highlighted underperforming regions and warehouses
Enabled better decision-making through clean and reliable data
